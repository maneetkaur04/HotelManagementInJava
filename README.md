# Hotel Management System (Java GUI)


A GUI-based Hotel Management System developed using Java.
This desktop application helps hotel staff manage room bookings, customer details, and room records efficiently through a user-friendly graphical interface.

📌 Project Overview

The Hotel Management System is designed to automate basic hotel operations such as:

- Staff authentication

- Room booking

- Updating room records

- Viewing room details

- Deleting bookings

The system provides a simple and intuitive interface for hotel staff to perform daily management tasks.

🖥️ Features
🔐 1. Staff Login

- Secure login system for hotel staff.
  
- Prevents unauthorized access.
  
- Only authenticated users can manage room records.

🛏️ 2. Book a Room

- Add new customer booking.

- Assign available room.

- Store customer and room details.

❌ 3. Delete Room Booking

- Remove customer booking details.

- Free up room for future bookings.

📋 4. View Room Details

- Display all booked rooms.

- View customer details associated with rooms.

✏️ 5. Update Room Details

- Modify customer or room information.

- Keep records up to date.

🛠️ Technologies Used

- Java (Core Java)

- Java Swing / AWT (for GUI)

- SQL

- MySQL (Database)

- JDBC (Java Database Connectivity)

- Object-Oriented Programming Concepts

🏗️ System Architecture

The project follows a structured approach:

GUI Layer → Handles user interaction.

Business Logic Layer → Manages room operations and validations.

Data Handling Layer → Stores and retrieves room/customer data.
